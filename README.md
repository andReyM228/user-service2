# User-Service

## Description
`user-service` manages Telegram bot user data. It stores and processes information about users, including cars and balances.

## Technologies
- Go
- MongoDB
- Docker

## Architecture
Uses REST API to interact with `tg-service` and `tx-service`.

## Launch
```bash
docker-compose up
```
